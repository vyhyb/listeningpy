listeningpy.gui.adaptive

This module contains the Adaptive class, which represents a basic Adaptive test window for convolution-based processing. It provides functionality for playing audio files, recording responses, and moving to the next set of sentences.

Attributes
  • parent (ctk.CTk): The parent customtkinter widget.
  • sets (pd.DataFrame): A DataFrame carrying the prepared randomized sets in the following format: | set_id | id | path | transcription | |--------|----|------|---------------| |set_id |id |path |transcription | This DataFrame can be obtained from the routines of the listeningpy.set_preparation module (adaptive_combination, randomization). This attribute also stores the responses, number of clicks per iteration, and time between the first and last clicked button.
  • irs (pd.DataFrame): A DataFrame containing the item response theory parameters.
  • set_id (str, optional): The ID of the set to be used. If None, all sets will be used.
  • initial_difficulty (int, optional): The index of the initial difficulty level. Default is -1.
  • processing_func (Callable, optional): The processing function to be used. Default is straight.
  • args, *kwargs: Additional arguments and keyword arguments associated with the processing function.

 58                master: ctk.CTk, 
 59                sets: pd.DataFrame,
 60                irs: pd.DataFrame,
 61                set_id: str = None,

 63                processing_func : Callable = straight,

 66            """Initialize the AdaptiveGUI class.
 67
 68            Parameters
 69            ----------
 70            master : ctk.CTk
 71                The master widget.
 72            sets : pd.DataFrame
 73                The DataFrame containing the sets.
 74            irs : pd.DataFrame
 75                The DataFrame containing the item response theory parameters.

 93                self.current_set = self.sets

 98            self.kwargs = kwargs

170    def play(self):
171        """
172        Plays the audio file with adaptive settings.
173
174        This method reads the impulse response (IR) file based on the current difficulty level,
175        and then plays the audio file with the specified processing function and additional arguments.
176
177        Args:
178            None
179
180        Returns:
181            None
182        """
183        ir, fs_ir = read(self.irs['path'].iloc[self.difficulty], always_2d=True)
184        print(self.kwargs.keys())
185        gui.play_click(
186            self.current_set.loc[self.progress.get(), "path"],
187            button=self.button_play,
188            next_buttons=[self.segmented_button, self.button_next],
189            processing_func=self.processing_func,
190            in2 = ir,
191            fs_in2 = fs_ir,
192            **self.kwargs)
193
194    def toggle_segmented(self, value):
195            """Toggle the segmented button based on the given value.
196
197            Parameters
198            ----------
199            value : str
200                The value to determine the state of the segmented button.
201                Possible values are 'Wrong' and 'Correct'.
202            """
203            if value == 'Wrong':
204                self.segmented_button.configure(
205                    selected_color=('#eb0000', '#8c0000'),
206                    selected_hover_color=('#8c0000', '#710000')
207                    )
208            elif value == 'Correct':
209                self.segmented_button.configure(
210                    selected_color=('#00eb00', '#007000'),
211                    selected_hover_color=('#007000', '#005000')
212                    )
213            
214    def reset_buttons(self):
215        """Reset the state of the buttons.
216
217        This method disables the segmented_button and button_next buttons.
218
219        """
220        self.segmented_button.configure(state=ctk.DISABLED)
221        self.button_next.configure(state=ctk.DISABLED)
222
223    def move_to_next(self):
224        '''Stores response to self.current_set dataframe and moves
225        to the next set.
226        '''
227        row = self.progress.get()
228        cols = ["answer", "ir_id"]
229        self.current_set.loc[row, cols] = [self.choice.get(), self.irs["id"].iloc[self.difficulty]]
230        
231        if self.choice.get() == "Correct":
232            self.current_set.loc[row, cols] = [1, self.irs["id"].iloc[self.difficulty]]
233            self.difficulty = self.difficulty + 1
234            self.progress.set(self.progress.get()+1)
235        elif self.progress.get() != 0:
236            self.current_set.loc[row, cols] = [0, self.irs["id"].iloc[self.difficulty]]
237            self.difficulty = self.difficulty - 1
238            self.progress.set(self.progress.get()+1)
239        else:
240            self.difficulty = self.difficulty - 1
241
242        print(f"Difficulty: {self.difficulty}\nSentence: {self.progress.get()}")
243        self.reset_buttons()
244        if self.progress.get() > self.last_idx:
245            self.master.responses = pd.concat(
246                [self.master.responses, self.current_set]
247            )
248            self.master.switch_frame(self.master.intermediate_frame)
249            return
250        self.progress_label_var.set(f"{self.progress.get()+1}")
251        self.current_sentence.set(self.current_set["transcription"].iloc[self.progress.get()])
252        # logging.info(self.current_set)
253    
254    def load_sentences(self):
255        """Load the sentences from the given path.
256        """
257        self.current_set = self.sets[self.sets['set'] == self.set_id].reset_index(drop=True)
258        print("===Current set:===", self.current_set)
259        self.progress.set(0)
260        self.progress_label_var.set(f"{self.progress.get()+1}")
261        self.difficulty = self.irs.index[self.initial_difficulty]
262        self.current_sentence.set(self.current_set["transcription"].iloc[self.progress.get()])
